العسكري و للمواطن

      المواطن

    • يجب على المواطنين توقف المركبة بعد الطلق على ( 4 كفرات )᲼᲼᲼

      يسمح للمواطن ( بالهروب ركض ) بعد نزوله من المركبة وقت المطاردة᲼᲼

      يمنع من المواطن الهروب بالمركبة ( قفز على البحر )᲼

      يمنع نهائيا قتل ( الموظف الرسمي داخل السيرفر ) سواء كان داخل أو خارج من المدينة في حال شغله

      يمنع من العسكري توجهة السلاح عليك ( داخل المدينة الأولى ) الأ في حال وجود بلاغ لديك

      يسمح ( قتل داخل المدينة الأولى ) بستعمال العصيان أو السكاكين ويجب يكون فيه أسباب قبل القتل

      يسمح القتل ( خارج المدينة الأولى ) سواء عسكري أو مواطن

      يسمح أستخدام جميع الأسلحه عند ( بائع أو شاري الممنوعات ), موقع الذيب

      يمنع أستعمال ( سلاح السنايبر ) داخل المدينة الأولى وموقع الذيب بشكل نهائي سواء عسكري أو مواطن

      العسكري

    • يمنع القتل ( داخل المدينة الأول ) سواء كان عسكري أو مواطن أو البوتات

      يمنع دخول أي وظيفة من الوظايف وعدم الأشتغال بالوظيفة لمدة ( ثلاث أيام ), سيتم محاسبتك, فيجب عليك خروج من الوظيفة᲼

      يسمح القتل ( خارج المدينة الأولى ) سواء عسكري أو مواطن

      يسمح على العسكري بستعمال ( سلاح الكهرب ) لأيقاف المواطن أو لتهديده داخل القسم

      يمنع أستعمال ( سلاح السنايبر ) داخل المدينة الأولى وموقع الذيب بشكل نهائي سواء عسكري أو مواطن

      يمنع من العسكري قتل المواطنين داخل ( المدينة الأولى )

      يبدأ العسكري ( بالطلق على كفارات ) مركبات المواطنين خلال ( 80 ثانية ) مع المطاردة᲼

      يبدأ العسكري ( بالصدم الأجباري ) على مركبة المواطنين خلال ( 30 ثانية ) مع المطاردة᲼᲼

      عند هروبك من الشرطة ركض, سيتم الطلق عليك تحذير للوقوف, وفي حال عدم وقوفك, سيتم الطلق عليك قتلا᲼

      يمنع من العسكري ( الصدم على المواطنين ) دون أسباب أو دون مطاردة ( داخل المدينة الأولى )᲼᲼᲼᲼᲼

      يمنع من العسكري مطاردة المواطنين دون تشغيل ( السفاتي )᲼
